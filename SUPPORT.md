## Before Asking for Help

Please check:

1. **Is it a known issue?** Search [existing issues](https://github.com/yafatek/mxp-py/issues)
2. **Is it documented?** Check the [README](README.md)
3. **Is it a protocol question?** See the [MXP specification](https://github.com/yafatek/mxp-protocol/blob/main/SPEC.md)

## When Reporting Issues

Please include:

- **Python version**: `python --version`
- **Package version**: `pip show mxp`
- **Operating System**: e.g., "macOS 14.0", "Ubuntu 22.04"
- **Code sample**: Minimal reproducible example
- **Error messages**: Full error output
